About Project FortiDragon

Forged in education. Focused on protection.
Project FortiDragon: Empowering cybersecurity learning through real-world vulnerability tracking.

Project FortiDragon is a Flask-based web application created as part of an advanced programming course. It demonstrates how web development and cybersecurity concepts can work together through a simple vulnerability disclosure and tracking system.

The goal of this project is to give students and developers hands-on experience building a secure web application that manages vulnerability reports, supports user authentication, and uses a database to store submissions. It reflects a practical approach to learning cybersecurity principles through real code.

  • Users can create accounts and log in securely.
  • Authenticated users can submit vulnerability reports.
  • Each report can include a title, description, severity, and mitigation steps.
  • Admins or reviewers can view and manage the submitted reports.

  • Backend: Flask (Python web framework)
  • Database: SQLite for local data storage
  • Templates: Jinja2 (Flask’s built-in templating system)

Future versions of Project FortiDragon could connect to external APIs such as the National Vulnerability Database (NVD), VirusTotal, or Jira to expand automation and threat intelligence capabilities.

Created by Jamie Smith as part of the Advanced Programming with Python course.
Inspired by academic work at Hutchinson Community College and Fort Hays State University.
